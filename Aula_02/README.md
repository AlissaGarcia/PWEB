# Atividades de 1 à 5

Unidade 1 – PHP Procedural e Web  
Período: 28/03/2026 a 04/04/2026  

## --Atividade1:

Sistema de Classificação Acadêmica

Sistema desenvolvido para receber a nota de um aluno e informar sua situação final.

Regras:
- Nota >= 7: Aprovado
- Nota >= 5 e < 7: Recuperação
- Nota < 5: Reprovado

Funcionalidades:
- Uso de if/else
- Uso de laço de repetição
- Uso de função
- Saída em HTML

---

Campos:

Funcionalidades:
- Método POST

- Exibição dos dados
- Mensagem de erro
